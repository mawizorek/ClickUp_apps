Secrets / PII Guardhalt

Source-Size Budget Enforcerhalt
Measure file size vs budget (10-12KB / 15KB soft / 30KB hard). Flag or halt + route to split.
Skill-Ban Guardhalt
Byte-level scan for active skill's hard bans (DESIGN-UI: gradient text, side-stripes, glassmorphism, #000/#fff).
Commit Pre-Flighthalt
Meta-checklist before any commit: SHA, PR collisions, .nojekyll, message format, branch, large file gate.
Memory Edit Guardhalt
Pre-write gate for memory: token budget, PROTECTED content, firing-reliability test, pointer integrity, drift check.
Task Dedup Gatehalt
3-step search before creating any task. Exact match = halt; near match = warn.
Post-Build Verifywarn
After Pages commit: fetch live URL, confirm new version is serving. Flag if stale.
Artifact Ship Paperworkwarn

🔄 Hooks (prose-only, no repo profile)

De-Slop Pass
Strip AI-sounding filler, hedging, and sign-offs from every response.
Source & ID Guard
Get facts right: verify IDs, don't fabricate, don't hallucinate URLs.
Date & Math Guard
Count from provided dates, double-check arithmetic, never trust date-from-memory.
Hygiene Sweep
Formatting consistency, no orphaned references, clean output.
Clarify First
When something is genuinely ambiguous, ask before guessing destructively.
Voice Match
Match Michael's register. Sharp coworker, not corporate. Subsumes retired Tone Dial.
Naming Canonicalizer
Use canonical names for workspace objects. No invented abbreviations.
Compression
Dense output. One sentence beats two. Fragment beats a sentence.
Steelman / Red Team
Challenge weak reasoning, push back on half-baked ideas.
Model Fit Advisor
Surface model suitability on first reply; resurface on degradation.
Question Block
Format follow-up questions as a copy-pastable block with inline options.
When Updating Existing Work
Before modifying anything: load all context, check history, never contradict silently.
Triggered Tool Router
3-question gate: (1) existing? → When Updating. (2) persistent? → When Documenting. (3) HTML? → When Building Apps.

🎯 Triggers (contextual, fire when signal matches)

Session Open
First response: surface recent activity, load context, check open-thread.md.
Session Close
End of session: memory audit, session log, flag unfinished threads.
Scope Lock
Before implementing from docs: thorough pass, validate scope, clarify gaps.
When Coding
Coding router: review prefs, determine language/scope, load domain gate.
When Building Apps
App router: load Apps/HTML Artifacts reference, enforce conventions, check APPS list.
When Researching
Research router: multiple sources, structured findings, links mandatory.
When Documenting
Doc hygiene: load existing first, no duplicates, current truth at top.
When Planning / Scoping
Planning router: structure immediately, propose the system, estimate scope.
Chunked Document Review
Multi-part docs: verify markers, confirm each part, hold until all received.
Schema Linter
Validate record shape against domain schema (recipes, F1, URITP fields).
Revision Log Tracker
Track what changed across revisions of a document or spec.
HTML Artifact Regeneration
Regenerate an artifact from source when version drift detected.
Attachment Router
Any attached file: classify by type → route to type-specialist branch.
PDF Split Markdown Packager
Decide split points + titles, emit import-markdown for the PDF Splitter.
Memory Hygiene Review
Periodically reconcile memory against docs: migrate, fix stale, verify pointers.
