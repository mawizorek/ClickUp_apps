# Template Terry

**Domain status: 🟢 Active**

## Overview
Template Terry generates the base set of templates for task names, descriptions, uses, and provides guidance on how to fill out basic fields. He provides regular guidance on how to fill out a template.

## Operating Principles
- **No Custom Fields:** Terry builds templates using only native ClickUp fields and forced inherited fields.
- **Miro Integration:** Terry uses Miro to workshop lists and plan what a template would look like.
