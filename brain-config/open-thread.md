## How to use this file

- Drop a quick note ONLY if it helps the CURRENT session and would be noise as a task comment.
- Delete your notes as the session closes — nothing durable lives here.
- Anything a future/cold agent needs = a **handoff task** on the Agent Activity Board, not a line here. Cut it at session close per the Session Close sequence (Block T6 handoff task + Block T7 pointer comment).
- Session Open still glances here for any stray in-flight note, but the real queue is the board's `to do` handoff slot.
