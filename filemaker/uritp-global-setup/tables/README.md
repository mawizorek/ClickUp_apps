# Tables

One file per table, mirroring *Manage Database → Tables*. Machine mirror lives in `../schema/tables.json`; each `.md` is the human-editable per-table doc. Keep `_index.json` in sync on every change.

## Format

Header (Role · Status · App) → description → **Fields** table → **Relationships** → **Open Items** → **Changelog**.

See [../../DOCUMENTATION-STANDARD.md](../../DOCUMENTATION-STANDARD.md) for the full standard.

> Config spine, target-state. PRODUCTIONS is NOT here (builder-owned, DG-001). Staff_Positions is the layer graduated from uritp-people (DP-006).
