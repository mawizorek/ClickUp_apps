PNG in, PNG out, actually smaller. Re-saving a PNG returns the same bytes because PNG is lossless. To shrink it Squash reduces the color palette (median-cut quantization), so DEFLATE packs it tighter, real byte reduction, transparency intact. It binary-searches the palette down until you're under cap, then downscales only if it has to. WebP/JPEG go smaller still by searching encoder quality; pick those when you don't need a literal PNG.
