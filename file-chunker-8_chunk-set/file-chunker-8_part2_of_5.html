File Chunker for AI Review

Split a large file into ordered chunks in its native format, then ship one self-instructing index that walks a receiving agent through them. With a hosted base URL, the index becomes self-navigating: the agent fetches each chunk itself on your "proceed", pausing and confirming after every part, and stops dead after the last one instead of acting on its own.

The self-navigating gated walk

  1. Host the chunk files at your base URL, then click Download set (or Index only) to grab the matching index.
  2. Hand the index file (or its URL) only to a fresh agent. It reads the index, confirms it understands the rules, and stops. It now has every chunk's full URL.
  3. Say "proceed". The agent fetches chunk 1 from its URL itself, verifies sequence + UUID, confirms completeness in chat, then stops and waits.
  4. Say "proceed" again for the next chunk. One fetch per "proceed", never reading ahead. Repeat through the set.
  5. After the final chunk the agent always pauses: reading every part is not permission to act. It waits for your explicit instructions before any rebuild or action.
Numbering is flat sequential (e.g. 3/7). Grouped/sectioned numbering (per portal/module) is a known future option, not active yet.
