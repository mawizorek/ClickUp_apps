The 3-state access layer

Open access: open

App shows to everyone with the link. Fully public.

Gated access: gated

The smokescreen you just cleared. A per-app code lifts the cover for the session. Default posture.

Down access: down

Cover with no way through. The hard kill switch: flip it, one-line commit, every copy goes dark.
